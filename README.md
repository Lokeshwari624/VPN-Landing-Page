
# VPN Landing Page

A responsive VPN service landing page built entirely using **HTML & CSS** in **VS Code** .
This project demonstrates practical coding skills and responsive web layout techniques.

---

## Features

- Fully responsive layout for desktop, tablet, and mobile  
- Header with navigation menu  
- Hero section with call-to-action (CTA)  
- Features section highlighting VPN benefits  
- Pricing / Plans section  
- Footer with contact info and social links  
- Smooth scrolling between sections  

---

## Technologies Used

- HTML5  
- CSS3  
- VS Code (for coding and file management)
